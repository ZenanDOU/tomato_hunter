# 🍅 Tomato Hunter 番茄猎人

> 把拖延变成怪物，把专注变成狩猎。

Tomato Hunter 是一款游戏化番茄钟效率工具。你的每一个拖延任务都会召唤一只拖延怪物占领番茄农场——而你，是被召唤来击败怪物、拯救番茄的勇者。

## ✨ 核心特色

**🐉 怪物生态系统**
- 5 个栖息地（齿轮工坊、枯竭画廊、遗忘图书馆、荒废花园、迷雾沼泽）
- 15 种怪物，基于真实食物链设计（小型猎物 → 中型捕食者 → 顶级掠食者）
- 每种怪物有独特的种族特性、专属身体部位、像素画形象

**⚔️ 狩猎即专注**
- 创建任务 → 侦查敌情（生成怪物）→ 制定策略 → 出击（番茄钟）→ 击败怪物 → 拯救番茄
- 互动式准备阶段：确认目标、记录策略、想好再动手
- 极简专注浮窗：大号倒计时 + 任务名 + 策略提示，减少干扰

**🧩 怪物拆分**
- 大任务（>4 番茄）可拆分为怪物的身体部位
- 每个种族有独特的部位结构（蟒 → 蛇首/蛇躯/蛇尾，蛾 → 双翼/胸腔/触角）
- 部位提示帮你思考任务分解方式（核心难点 / 主体执行 / 收尾交付）

**🛡️ 装备系统**
- 武器改变番茄钟时长（标准短刀 25min / 长弓 50min / 匕首 15min）
- 护甲提供容错机制（暂停时长、短暂中断）
- 消耗品：烟雾弹（暂停）、时光沙漏（延长专注）、猎人直觉（掉落翻倍）

**🍅 番茄收容所**
- 每次击败怪物都会拯救一颗番茄
- 收容所记录你的战斗历程、策略笔记、反思记录
- 番茄墙：看着被救回的番茄越来越多，成就感满满

## 🏗️ 技术栈

- **前端**: React 19 + TypeScript + Tailwind CSS v4
- **桌面框架**: Tauri 2 (Rust)
- **数据库**: SQLite (tauri-plugin-sql)
- **状态管理**: Zustand
- **像素字体**: Zpix
- **像素画**: Canvas API 程序化生成

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 开发模式（前端 + Tauri）
npm run tauri dev

# 构建生产版本
npm run tauri build
```

需要安装 [Rust](https://rustup.rs/) 和 [Node.js](https://nodejs.org/) 22+。
